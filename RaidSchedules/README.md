# RaidSchedules

A visual schedule of server, zone, and game events with icons for each raid boss and activity. Great for planning your day around farm windows, world bosses, and naval events.

## Preview

![RaidSchedules](image/imagename.png)

## Features

- Categorized event lists (server events, zone events, game events).
- Icon-driven UI with bespoke art for major bosses (Kraken, Leviathan, Anthalon, etc.).
- Reset timer indicators.
- Draggable window with persistent position.

## Usage

Click the toggle button to open the schedule. Event times are configured in `TimesAndIcones.lua`.
