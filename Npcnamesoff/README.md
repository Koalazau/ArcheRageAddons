# Npcnamesoff

A simple toggle for hiding (and re-showing) NPC nameplates. Cleans up the screen during cinematic moments, screenshots, or busy hubs.

## Preview

![Npcnamesoff](image/imagename.png)

## Features

- One-click button to toggle NPC names on/off.
- Lightweight — no background polling or extra dependencies.

## Usage

Click the toggle button to hide or show NPC names. Reposition the button by dragging it.
