# PassiveTracker

Tracks the buffs you care about as icons on screen so you can monitor passives, skill cooldowns, and procs at a glance.

## Preview

![PassiveTracker](image/imagename.png)

## Features

- Up to 7 tracked buff icons displayed side by side.
- Remembers icon window position across sessions.
- Captures skill cooldown state and previous skill activations for accurate tracking.
- Configurable via `config.lua`.

## Usage

Edit `config.lua` to define which buffs/skills to monitor. The tracker window can be dragged anywhere on screen and its position is saved.
