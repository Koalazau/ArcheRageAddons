# ExpirationChecker

Tracks the duration of important buffs on your character and warns you when one expires. Useful for keeping uptime on consumables, scrolls, and other timed effects.

## Preview

![ExpirationChecker](image/imagename.png)

## Features

- Displays icons for the buffs you choose to track.
- Highlights expired buffs with an "expired" texture overlay.
- Movable on-screen window that remembers its position.

## Usage

Drag the window to reposition it. Expired buffs are flagged visually so you can quickly refresh them.
