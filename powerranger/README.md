# powerranger

A one-click toggle button for the in-game **Custom Clone Mode** option. Saves you a trip into the settings menu every time you want to switch.

## Preview

![powerranger](image/imagename.png)

## Features

- Single floating button that flips Custom Clone Mode on/off.
- Draggable, with position persisted to disk.
- Visual state changes to reflect current toggle.

## Installation

1. Copy the `powerranger` folder into your ArcheRage `addon` directory.
2. Launch the game — the addon loads automatically.

## Usage

Click to toggle. Drag to relocate the button.
