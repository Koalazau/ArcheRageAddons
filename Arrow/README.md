# Arrow

An on-screen directional indicator that always points to your current target. When the target is off-screen the arrow points from the center of your view toward them; when they are on-screen it floats above their head and points down.

## Preview

![Arrow](image/imagename.png)

## Features

- Tracks your selected target in real time.
- Off-screen mode rotates the arrow toward the target's direction.
- On-screen mode anchors the arrow above the target pointing down.
- UI-scale aware so positioning stays consistent at any resolution.

## Installation

1. Copy the `Arrow` folder into your ArcheRage `addon` directory.
2. Launch the game — the addon loads automatically.

## Usage

Select any unit. The arrow appears as long as you have a valid target and disappears when no target is selected.
