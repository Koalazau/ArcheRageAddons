# RaidNotification

Pops a notification window for upcoming raid and server events so you never miss a spawn. Pairs naturally with **RaidSchedules** for a full schedule + alert workflow.

## Preview

![RaidNotification](image/imagename.png)

## Features

- Timed pop-up notifications for raid / world events.
- Per-event cooldowns to avoid spam.
- Status bar countdowns for active alerts.
- Toggle button to enable / disable notifications quickly.

## Usage

Event timings are sourced from `Times.lua`. Reposition the toggle button by dragging it; notifications appear automatically when their event approaches.
