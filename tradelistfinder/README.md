# tradelistfinder

Browse housing for sale across continents and zones from inside the game. Lists houses, prices, types, and sellers so you can shop without running across the map.

## Preview

![tradelistfinder](image/imagename.png)

## Features

- Continent and zone selectors to filter results.
- Tabulated columns: house type / size, seller, price.
- Paginated list with a draggable, position-saving window.
- Custom button icons.

## Usage

Click the toggle button to open the search window, pick a continent and zone, and review listed houses for sale.
