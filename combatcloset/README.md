# combatcloset

A gear / costume preset manager. Build loadouts for different situations — PvE, PvP, Trading, Packs, Sleep — and swap between them in one click.

## Preview

![combatcloset](image/imagename.png)

## Features

- Multiple preset slots with custom icons (PvP, PvE, Trade, Packs, Sleep, etc.).
- One-click equip from saved loadouts.
- Drag-and-drop UI built on combo boxes and labelled buttons.
- Customizable hotkeys via the in-game API.

## Usage

Open the closet window, configure presets for each situation, and switch presets with the dedicated icons or your bound hotkey.
